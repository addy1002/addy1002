## 🚀 Featured Projects

### 🎮 Super Mario on Kubernetes (EKS)
> Deployed a containerized Super Mario app on AWS EKS cluster using Terraform for infrastructure provisioning.

**Stack:** `AWS EKS` `Terraform` `kubectl` `EC2` `IAM`

- Provisioned EKS cluster with Terraform from scratch
- Configured node groups, IAM roles, and kubeconfig
- Debugged real-world state drift issues in Terraform

---

### 🔐 DevSecOps CI/CD Pipeline — Insure Me
> End-to-end CI/CD pipeline with integrated security scanning for a Java insurance application.

**Stack:** `Jenkins` `SonarQube` `Docker` `Trivy` `AWS S3` `DockerHub` `Maven`

- Automated build → code quality → artifact upload → container deploy
- Integrated Trivy for Docker image vulnerability scanning
- Quality Gate with SonarQube to block bad code from deploying

---
